# Ratings & Reviews System

A full-stack web application. 
Users can log in and submit ratings and reviews for products. Each user is allowed to review a product only once.

---

## Features
- User authentication (Email & Google) using Firebase
- Product listing with images and descriptions
- Submit rating (1–5) and review text
- One review per user per product
- Reviews stored and fetched from MySQL database
- Clean and professional UI

---

## Tech Stack
- **Frontend:** React.js, CSS
- **Authentication:** Firebase Auth
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **API Communication:** Axios

---
